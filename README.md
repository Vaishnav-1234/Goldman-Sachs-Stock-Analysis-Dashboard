# Goldman Sachs Stock Analysis Dashboard

Built an end-to-end data analysis project using historical stock data.

## Tools
Excel | PostgreSQL | Tableau

## Key Work
- Cleaned and validated raw stock data
- Fixed date format issues and anomalies
- Created KPIs: Return, Volatility, Price Trend
- Built interactive Tableau dashboard

## Insights
- Strong long-term growth
- Stable returns (~0.24%)
- Volatility decreasing over time

## Output
Interactive dashboard for performance & risk analysis
