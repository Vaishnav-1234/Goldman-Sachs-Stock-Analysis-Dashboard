# 📊 Goldman Sachs Stock Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing historical stock data of Goldman Sachs to derive meaningful insights on price trends, returns, and volatility. The goal was to transform raw financial data into a clean, structured, and interactive dashboard for data-driven decision-making.

---

## 🎯 Objective
- Understand long-term stock performance
- Analyze daily returns and volatility (risk)
- Identify patterns, trends, and anomalies in stock behavior
- Build a professional dashboard for stakeholders

---

## 📂 Dataset
The dataset contains historical stock data with the following fields:
- Trade Date  
- Open Price  
- High Price  
- Low Price  
- Close Price  
- Adjusted Close  
- Volume  

---

## 🛠 Tools & Technologies
- **Excel** → Data cleaning & preprocessing  
- **PostgreSQL** → Data validation & querying  
- **Tableau** → Data visualization & dashboard creation  

---

## 🔧 Data Cleaning & Preparation
Key challenges and fixes:
- Fixed incorrect date formats (dd/mm/yyyy vs mm/dd/yyyy issue)
- Identified and corrected swapped OHLC columns
- Removed duplicate records and anomalies
- Handled extreme spikes and unrealistic values in price & returns
- Created calculated fields:
  - Daily Return
  - Volatility (OHLC-based)
  - Moving averages

---

## 📊 Dashboard Features

### 🔹 KPI Section
- Latest Price  
- Average Return  
- Volatility  
- Highest Price  
- Lowest Price  

### 🔹 Visualizations
- 📈 **Price Trend Chart** → Long-term stock performance  
- 📉 **Daily Return Chart** → Profit/Loss fluctuations  
- ⚡ **Volatility Chart** → Risk analysis over time  

---

## 📈 Key Insights
- Stock shows a strong long-term upward trend  
- Average daily returns are stable (~0.24%)  
- Volatility has decreased over time, indicating improved stability  
- Major spikes correspond to market events (e.g., financial crises)  

---

## 💡 Learnings
- Data cleaning is the most critical step in analysis  
- Incorrect data (dates, values) can completely distort insights  
- Visualization should focus on clarity and business understanding  
- Combining tools (Excel + SQL + Tableau) improves workflow efficiency
